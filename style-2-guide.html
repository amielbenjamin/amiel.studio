theme-clean · reference

Quiet Editorial

The second theme. Same content model, same DOM as theme-bold — the shift is purely token and motion. A monochrome system built on off-white and charcoal, hairline borders, generous line-height, and a single spot of colour that enters only as soft light, never as a fill. Every value below is read from the live [data-theme="clean"] tokens in css/style.css.

01 — Colour

A neutral grayscale base. The legacy --pink* slots resolve to ink and grey, so every inherited reference reads monochrome — no hot pink, no clay. Colour arrives through one gradient only (see 04).

03 — Accent light

The only colour in the system. The existing brand pink (#ff80cb) survives solely as a soft radial wash — --glow — layered behind surfaces at low opacity. It is light, not paint: never a border, fill or text colour.

06 — Motion & token reference

150–250ms ease-out for UI. No bounce, no big parallax. In JS, themeClean swaps the reveal easing from back.out(1.5) to power2.out.
